## Contributing to Breezely

### Setup de desarrollo

1. Clonar repositorio
2. Instalar dependencias backend: `pip install -r backend/requirements-dev.txt`
3. Instalar dependencias frontend: `flutter pub get`

### Convenciones

- **Backend:** Python con PEP 8, type hints obligatorios
- **Frontend:** Dart con effective_dart
- Commits en español o inglés, descriptivos

### Pull Request Process

1. Crear rama desde `main`: `git checkout -b feature/nombre-feature`
2. Hacer cambios con tests
3. Ejecutar `pytest` y `flutter test`
4. Crear PR con descripción clara

## Código de Conducta

Sé respetuoso y constructivo.
